## �️ INSTALACIÓN DE HERRAMIENTAS LaTeX

Antes de empezar, todos los miembros del equipo deben instalar estos programas para compilar el documento LaTeX:

### 1️⃣ Instalar Strawberry Perl

**¿Qué es?** Intérprete de Perl necesario para que latexmk funcione correctamente.

**Pasos:**
1. Descarga desde: [Strawberry Perl](https://strawberryperl.com/)
2. Ejecuta el instalador `.msi`
3. Sigue la instalación por defecto (Next, Next, Finish)
4. Reinicia tu computadora

**Verificar instalación:**
```powershell
perl -v
```
Debe mostrar la versión de Perl instalada.

### 2️⃣ Instalar MiKTeX

**¿Qué es?** Distribución de LaTeX que contiene todo lo necesario para compilar documentos.

**Pasos:**
1. Descarga desde: [MiKTeX](https://miktex.org/download)
2. Ejecuta el instalador `.exe`
3. En la instalación, selecciona:
   - **Installation folder**: `C:\Program Files\MiKTeX`
   - **Preferred paper**: A4
   - **Install missing packages on-the-fly**: ✅ SÍ (importante)
4. Completa la instalación
5. **IMPORTANTE**: Abre "MiKTeX Console" desde inicio y actualiza paquetes:
   - Click en "Updates" → "Get Updates" → "Update"

**Verificar instalación:**
```powershell
pdflatex --version
```
Debe mostrar la versión de pdflatex.

### 3️⃣ Instalar Extensión LaTeX Workshop en VS Code

**¿Qué es?** Extensión para editar y compilar documentos LaTeX directamente en VS Code.

**Pasos:**
1. Abre VS Code
2. Ve a Extensions (Ctrl + Shift + X)
3. Busca "LaTeX Workshop"
4. Instala la extensión por **James Yu** (la oficial)
5. Reinicia VS Code

**Configuración (Opcional pero recomendada):**
En VS Code, abre Configuración (Ctrl + ,) y busca "latex workshop":
- **Latex-workshop: Latex recipes**: Asegúrate que esté configurado para `pdflatex`
- **Latex-workshop: View pdf on double click**: Activa para previsualizar

**Verificar:**
- Abre `documentacion/main.tex`
- Deberías ver opciones para compilar en la esquina superior derecha

### 4️⃣ Clonar el Repositorio

```powershell
# Navega a donde quieras guardar el proyecto
cd "C:\Ruta\Del\Proyecto"

# Clona el repositorio
git clone https://github.com/BryanGtmk/Proyecto_BD.git

cd Proyecto_BD
```

## 📖 COMO COMPILAR EL DOCUMENTO LATEX

### Opción 1: Desde VS Code (Recomendado)

1. Abre `documentacion/main.tex`
2. Haz clic en **"Build LaTeX project"** (ícono verde en la esquina superior derecha)
3. Espera a que termine la compilación
4. Verás el PDF en la vista previa

### Opción 2: Desde Terminal PowerShell

```powershell
# Navega a la carpeta documentacion
cd documentacion

# Compila (primera vez puede tardar más)
pdflatex -interaction=nonstopmode main.tex

# Si hay cambios en bibliografía, corre:
bibtex main.aux

# Compila de nuevo
pdflatex -interaction=nonstopmode main.tex
```

**Resultado:** Se genera `main.pdf` en la carpeta `documentacion/`

### Opción 3: Usar latexmk (Automático)

```powershell
cd documentacion
latexmk -pdf main.tex
```
